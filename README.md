# 💌 팬레터 소개
 뮤지컬 팬레터 팬페이지 입니다.

판매자는 상품을 구매할 수 없으며,<br> 
상품의 정보를 등록 및 수정을 통해 상품을 판매할 수 있습니다.<br>

구매자는 호두마켓을 통해 상품의 세부정보를 확인하고 구매할 수 있습니다.
